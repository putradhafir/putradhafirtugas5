01. Pengenalan Sealed Class

✦ Pengertian Sealed Class

  • Sealed class adalah jenis class khusus dalam bahasa pemrograman Dart yang membatasi pewarisan (inheritance) hanya dalam satu file yang sama.

✦ Contoh Implementasi

// Nama  : PUTRA DHAFIR NASHRULLAH
// NIM   : 251410009
// Kelas : SI2A

// Sealed class membatasi pewarisan hanya dalam satu file
sealed class ApiResponse {}

class Success extends ApiResponse {
  final String message;
  Success(this.message);
}

class Error extends ApiResponse {
  final String errorMessage;
  Error(this.errorMessage);
}

void handleResponse(ApiResponse response) {
  if (response is Success) {
    print("✅ Success: ${response.message}");
  } else if (response is Error) {
    print("❌ Error: ${response.errorMessage}");
  }
}

void main() {
  print("=" * 50);
  print("CONTOH SEALED CLASS");
  print("=" * 50);
  
  handleResponse(Success("Data berhasil diambil"));
  handleResponse(Error("Terjadi kesalahan pada server"));
}

02. Keuntungan menggunakan Sealed Class

✦ Pengertian Keuntungan Menggunakan Sealed Class

  • Sealed class adalah jenis class khusus dalam bahasa pemrograman Dart yang membatasi pewarisan (inheritance) hanya dalam satu file yang sama. Fitur ini diperkenalkan di Dart 3.0 untuk memberikan kontrol yang lebih ketat terhadap hierarki class.
// Nama  : PUTRA DHAFIR NASHRULLAH
// NIM   : 251410009
// Kelas : SI2A

class Animal {
  void speak() {
    print("Hewan bersuara");
  }
}

class Dog extends Animal {
  @override
  void speak() {
    print("🐕 Anjing: Woof! Woof!");
  }
}

class Cat extends Animal {
  @override
  void speak() {
    print("🐈 Cat: Meow! Meow!");
  }
}

void main() {
  print("=" * 50);
  print("CONTOH POLIMORFISME (METHOD OVERRIDING)");
  print("=" * 50);

  Animal animal = Animal();
  animal.speak();

  Dog dog = Dog();
  dog.speak();

  Cat cat = Cat();
  cat.speak();

  print("\n--- Polimorfisme dengan parent reference ---");
  Animal hewan1 = Dog();
  Animal hewan2 = Cat();
  hewan1.speak();
  hewan2.speak();
}

03. Polimorfisme Class

✦ Pengertian Polimorfisme Class

  • Polimorfisme adalah kemampuan dalam pemrograman di mana satu nama (metode atau antarmuka) bisa memiliki banyak perilaku yang berbeda. Tujuannya agar kode lebih efisien; Anda tidak perlu membuat banyak nama fungsi yang berbeda untuk tugas yang serupa.
// Nama  : PUTRA DHAFIR NASHRULLAH
// NIM   : 251410009
// Kelas : SI2A

// Final class tidak dapat diwarisi (di-extend)
final class Settings {
  String theme;
  int fontSize;

  Settings({required this.theme, required this.fontSize});

  void showSettings() {
    print("⚙️ Settings - Theme: $theme, Font Size: $fontSize");
  }
}

void main() {
  print("=" * 50);
  print("CONTOH FINAL CLASS");
  print("=" * 50);

  Settings settings = Settings(theme: "Dark Mode", fontSize: 16);
  settings.showSettings();

  // Jika ingin mencoba extends, akan error:
  // class WrongSettings extends Settings {} // ERROR: Cannot extend final class
}
